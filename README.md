## 📱 Cómo Instalar

### OPCIÓN 1: Descarga Directa (Recomendado)
```
1. Descarga el APK desde el enlace arriba
2. Envíalo a tu teléfono (email, WhatsApp, etc.)
3. Abre en tu teléfono
4. Tap en INSTALAR
5. ¡Listo! 🎉
```

### OPCIÓN 2: Compilar desde Código
```bash
# 1. Requisitos: Node.js v16+
# Descarga desde: https://nodejs.org/

# 2. Clonar
git clone https://github.com/SoyMoli/Downloader-app-by-Moli
cd Downloader-app-by-Moli

# 3. Instalar
npm install
npm install -g expo-cli eas-cli

# 4. Compilar
eas build --platform android --profile preview

# 5. Descargar cuando termine
```

## ❌ Problemas Comunes

### APK no instala
- Verifica Android 5.0+
- Libera espacio
- Desinstala versión anterior

### La app no descarga
- Verifica internet
- Comprueba permisos
- Reinicia la app

### Compilación falla
```bash
expo cache --clear
rm -rf node_modules
npm install
eas build --platform android --profile preview
```
