# 🎥 Downloader App by Moli

Universal video/audio downloader with integrated browser and multi-language support.

## ✨ Features

- 🌐 **Universal Downloader**
  - YouTube & YouTube Music
  - TikTok, Instagram, Facebook
  - Twitter/X, Twitch, Pinterest, Reddit
  - Vimeo, Google Drive, Google Docs
  - SoundCloud, Telegram

- 🌍 **Integrated Browser**
  - Built-in WebView browser
  - Quick download button
  - Navigation history
  - Bookmarks

- 🗣️ **Multi-Language Support**
  - Spanish, English, Portuguese
  - French, German, Italian
  - Russian, Chinese

- 📥 **Download Management**
  - Multiple quality options
  - Format conversion
  - Pause/Resume downloads
  - Download history
  - Custom storage location

- 🎨 **UI/UX**
  - Light/Dark theme
  - Notifications
  - Share downloads
  - Easy-to-use interface

## 🚀 Installation

### Prerequisites
- Node.js (v16+)
- Expo CLI: `npm install -g expo-cli`
- Android SDK / Xcode

### Setup

```bash
# Clone repository
git clone https://github.com/SoyMoli/Downloader-app-by-Moli
cd Downloader-app-by-Moli

# Install dependencies
npm install

# Start development
npm start

# Run on Android
npm run android

# Build APK
npm run build:apk
```

## 📱 Usage

1. Open app
2. Paste video URL or use integrated browser
3. Select quality/format
4. Download!

## 🔧 Tech Stack

- **Frontend**: React Native + Expo
- **Navigation**: React Navigation
- **Localization**: i18next
- **HTTP**: Axios
- **Storage**: React Native FS
- **UI**: React Native Vector Icons
